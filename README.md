# Telecom Customer Churn

1. Clone the repo

```bash
git clone https://github.com/siddharthborderwala/telecom-churn.git 
```

2. Go into the folder

```bash
cd telecom-churn
```

3. Setup virtual env by running

```bash
python3 -m venv .env/
```

4. Install all dependencies

```bash
pip install -r requirements.txt
```

5. Have fun!
